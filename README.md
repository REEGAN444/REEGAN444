## 📌 Projects

### 📷 Face Recognition System

A Computer Vision project that detects and recognizes faces using a webcam.

- **Tech Stack:** Python, OpenCV, NumPy, OS module
- **Features:**
  - Face dataset creation using webcam
  - Image storage for training
  - Real-time face detection and recognition
  - Uses Haarcascade algorithm
- **Workflow:**
  1. Run `facerecstore.py` to collect face data
  2. Train and store dataset images
  3. Run `faceRecognition.py` for real-time detection
  4. System recognizes faces from dataset
